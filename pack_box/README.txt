当年面试微信的第5到算法题

题目：
	仓库中有若干个箱子，现在需要打包发送至另外一个仓库。每个箱子的长、宽、高分别为L、W、H。当一个箱子A的长、宽、高分别小于另外一个箱子B的长、宽、高时，我们可以将A装入B中。为了尽可能减小运输体积，现请你设计一个函数int merge_boxes(const Box *boxes, size_t nbox, const Box *output_boxes, size_t *output_nbox)来尽可能将体积较小的箱子装进较大箱子中，使最终要搬的箱子总个数最小，并输出最终要运输的每一个最外面的箱子的大小。假定所有的箱子都可以任意翻转。

示例：
	输入为：[1,2,3] [2,3,4] [1,3,4] [3,4,5]
	输出为：[1,3,4] [3,4,5]

上述输入样例，答案好像错了，全都能放到[3,4,5]里。
